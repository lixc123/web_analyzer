# 文档目录结构

本目录包含项目的所有文档，按类型分类组织。

## 目录说明

### analysis/
分析报告和代码审查文档
- 代码完整性分析
- 硬编码分析和修复报告
- 清理报告
- 启动状态报告

### deployment/
部署相关文档
- 部署指南
- 生产环境配置指南
- 部署前检查清单
- 部署前审查报告

### guides/
使用指南和开发文档
- 组件使用指南
- 环境配置说明
- 前端解决方案指南
- 命令分类说明
- 修复总结

### tasks/
任务清单和待办事项
- AI分析任务清单
- Windows应用端抓包任务
- 生产环境待办事项
- 功能对齐任务清单

### testing/
测试相关文档
- 测试报告
- 验证报告
- 测试验证清单

### 其他文档
- `analysis_bundle_spec.md` - 分析包规范
- `mobile_pinning_handling.md` - 移动端证书固定处理

## 历史文档

项目根目录下的 `历史文档/` 目录包含早期版本的文档和已归档的文件。
